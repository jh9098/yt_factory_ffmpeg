# 이미지 크기 맞춤(Scale Mode) 가이드

## 왜 프리뷰와 렌더가 달라 보일 수 있나요?
쇼츠는 보통 9:16 비율(예: 1080x1920)을 사용합니다.
가로로 긴 이미지를 세로 영상에 맞출 때, `cover`를 쓰면 화면을 꽉 채우는 대신 바깥이 잘릴 수 있습니다.

## 모드 설명
- `전체 표시(잘림 없음=contain)`
  - 원본 이미지를 모두 보여줍니다.
  - 남는 공간은 여백(배경색)으로 채워집니다.
- `화면 채우기(일부 잘림=cover)`
  - 화면을 빈 공간 없이 꽉 채웁니다.
  - 비율이 다르면 상하 또는 좌우가 잘릴 수 있습니다.

## 예시 (9:16 쇼츠에서 가로 이미지 사용)
- 원본: 16:9 가로 이미지
- `contain`: 이미지 전체가 보이지만 위/아래 또는 좌/우에 여백이 생길 수 있음
- `cover`: 여백은 없지만 이미지 바깥 영역 일부가 잘릴 수 있음

## 이번 반영 내용
- 프리뷰와 렌더러가 같은 `scale_mode` 기본값/정규화 로직을 공유합니다.
- 타임라인 생성 시 각 아이템에 `scale_mode`를 명시적으로 저장합니다.
- 렌더 직전에 `scale_mode`, `crop_x/y/w/h`를 디버그 로그로 출력해 추적할 수 있습니다.
